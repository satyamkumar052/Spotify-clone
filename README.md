# 🎵 Spotify Clone

A responsive web-based music streaming interface inspired by Spotify. This project showcases a modern, dark-themed UI with an interactive music player and dynamic content layout.


## ✨ Features

- **Sidebar Navigation**
  - Home and Search options with icons
  - Your Library section for managing playlists
  - Quick action buttons (Create playlist, Browse podcasts)

- **Main Content Area**
  - Recently Played section with card-based layout
  - Featured Charts with multiple music chart options
  - Shows to Try featuring popular podcasts
  - Smooth scrolling and responsive card display

- **Music Player**
  - Album artwork display with song title and artist
  - Playback controls (Previous, Play/Pause, Next, Shuffle, Repeat)
  - Progress bar with current time and total duration
  - Volume control with slider
  - Additional features (Microphone, Queue, Share)

- **Design Elements**
  - Dark theme matching Spotify's aesthetic
  - Smooth hover effects and transitions
  - Font Awesome icons for UI elements
  - Material Icons integration
  - Mobile-responsive layout
